Turath-Tech 3D Scanner

Visualisation interactive du modèle 3D

Description

Ce modèle 3D a été obtenu à partir d'un système de numérisation automatisé utilisant :

  • Arduino UNO
  • Moteur pas à pas
  • Caméra USB
  • Python
  • Traitement 3D

L'utilisateur peut faire tourner le modèle, zoomer et l'explorer directement depuis le navigateur.
